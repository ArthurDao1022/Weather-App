#Weather App README 

Overview
This Weather App project demonstrates my proficiency in HTML, CSS, and JavaScript. The application is designed to provide real-time weather information in Fahrenheit for any city in the world using data retrieved from the OpenWeather API. I developed this application with the assistance of a YouTube tutorial by GreatStack.

Installation
To run the Weather App, you can either:

Access the live version online at https://github.com/ArthurDao1022/Weather-App

OR

Follow these steps to run it locally:

a. Download the project from the GitHub repository: https://github.com/ArthurDao1022/Weather-App

b. Install the Live Server extension by Ritwick Dey for your preferred code editor.

c. Right-click on the index.html file and select "Open with Live Server." This will launch the Weather App in your default web browser.

Usage
Upon opening the Weather App, you will be presented with a simple and user-friendly interface. You can follow these steps to check the weather for a specific city:

Enter the name of the city in the input field provided.

Click on the "Get Weather" button to fetch and display the weather information.

The app will show you the current temperature in Fahrenheit, along with other relevant weather details such as humidity, wind speed, and weather description.

To check the weather for another city, repeat steps 1-3.

Features
Real-time weather data: The Weather App utilizes the OpenWeather API to provide up-to-date weather information for any city worldwide.

User-friendly interface: The app offers a simple and intuitive interface, making it easy for users to check the weather without any hassle.

Temperature in Fahrenheit: The temperature is displayed in Fahrenheit to cater to users who prefer this unit of measurement.
